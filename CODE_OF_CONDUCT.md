# Code of Conduct

## My Pledge

This is a learning project focused on education and technical growth. All interactions should be:

- **Respectful** - Treat others with kindness
- **Constructive** - Focus on helping, not criticizing
- **Educational** - Remember this is a learning environment
- **Patient** - Everyone is learning at their own pace

## Scope

This applies to all project spaces: issues, pull requests, discussions, and any project-related communication.

## Standards

**Acceptable:**
- Asking questions about the code
- Suggesting improvements
- Reporting bugs
- Sharing related resources
- Constructive technical criticism

**Not Acceptable:**
- Personal attacks
- Harassment of any kind
- Trolling or inflammatory comments
- Publishing others' private information

## Enforcement

As the sole maintainer, I'll address any issues as they arise. Violations may result in:
1. Warning
2. Temporary ban from project
3. Permanent ban

## Contact

Report issues by opening an issue with the `conduct` label.
